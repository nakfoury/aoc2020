# aoc2020

Script that computes solutions for [Advent of Code 2020](https://adventofcode.com/2020).

## Usage
```
main.py [-h] [-d DAY] [-b]

optional arguments:
  -h, --help         show this help message and exit
  -d DAY, --day DAY  specify puzzle day <1...25>
  -b                 switch to part B
```
